# Git e GitHub

## Aprendendo comandos Git

### Comandos básicos

#### git init

~~~ bash
git init
~~~
- Comando para iniciar um repositório Git.

#### git config

~~~ bash
git conig user.name 'nome-do-usuario'

git config user.email 'email-do-usuario'
~~~
- Comando para configurar credencial de usuário.

#### git add

~~~ bash
git add nome-do-arquivo
~~~
- Comando para adcionar arquivos à área de preparação para commit.

#### git commit

~~~ bash
git commit -m 'menssagem de commit'
~~~
- Comando para adicionar um ponto na linha do tempo.
